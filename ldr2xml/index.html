Transform Sabrix XML

Clean, Rename & Redact your Sabrix XML files

How to use:

  1. In SQL Developer, select your records with the XML stored in a CLOB
  2. Use Export → Save as .ldr files using the Multiple files option
  3. Zip all the .ldr files and drop the ZIP here
📦

Drop your ZIP file here

or click to browse

This app:

  • Reads .ldr files that are inside ZIP
  • Pretty-prints the XML
  • Renames to invoicenumber.in.xml / invoicenumber.ou.xml
  • Redacts only in .in.xml files:
    • USERNAME → first 3 characters + ******
    • PASSWORD → REDACTED
